# Robotica_de_Desarrollo_Intro_a_ROS2Humble_Turtlesim
Implementación de control para una tortuga virtual en ROS 2 Humble usando Python y consola Linux. Incluye manejo manual con teclado, dibujo automático de iniciales, diagrama de flujo en Mermaid y organización clara del código, priorizando comprensión, replicabilidad y evolución del proyecto.
